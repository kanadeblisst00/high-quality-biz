## 描述

这个仓库记录我关注的一些关于逆向的公众号，不过因为我也不是每个公众号都看过，可能混进来一些干货不多的，如果你发现了可以反馈。

该仓库每天定时同步更新一次我的关注列表。另外，我正在下载这些公众号的全部历史文章，保留成html和pdf。到时候弄个网盘分享出来，并且每周或每两周更新一次。

如果你想自己下载公众号历史文章，可以看我写的[公众号下载助手](https://mp.weixin.qq.com/s/g0Tor1IxTOCIXNP5CCiOrQ)：`https://github.com/kanadeblisst00/biz-download-helper`

#### 关注公众号

如果你想关注某个公众号，你可以在公众号简介页面点击公众号名称会跳转到公众号二维码，扫码关注即可。如果你想批量全部关注，你可以下载`二维码`目录然后一个一个扫码关注里面的二维码。
